# Global-Video-Game-Sales-Dashboard-1980-2020
This project presents an interactive Video Game Sales Analysis Dashboard built using the Video Game Sales dataset. The dashboard explores global video game sales trends, genre popularity, platform performance, and regional sales distribution from 1980 to 2020.
The goal of this project is to provide clear insights into the video game industry using interactive data visualizations.

📂 Dataset
Dataset source:
👉 https://www.kaggle.com/datasets/gregorut/videogamesales

The dataset contains information about video game sales worldwide, including sales by region, platform, publisher, and genre. It includes over 16,000 games with sales greater than 100,000 copies.

Dataset Features
Column	Description
Rank	Ranking based on global sales
Name	Video game title
Platform	Gaming platform (PS2, Xbox, PC, etc.)
Year	Release year
Genre	Game genre
Publisher	Game publisher
NA_Sales	Sales in North America (millions)
EU_Sales	Sales in Europe (millions)
JP_Sales	Sales in Japan (millions)
Other_Sales	Sales in other regions (millions)
Global_Sales	Total worldwide sales (millions)

The dataset allows analysis of sales distribution across regions, genres, publishers, and gaming platforms.

📈 Dashboard Features
1️⃣ Overview Metrics

The dashboard includes key performance indicators:
Total Global Sales
Total Number of Games
Average Game Sales
Total Number of Publishers
These metrics provide a quick overview of the overall video game market.

2️⃣ Global Sales Trend by Year
A line chart visualizes how video game sales have changed over time.
Insights:
Sales steadily increased during the 2000s gaming boom.
The industry experienced peak sales around 2008–2009.
Sales decline in later years may reflect data reporting delays or digital distribution changes.

3️⃣ Sales by Genre

A bar chart shows the most popular game genres globally.
Key observations:
Action and Sports games dominate global sales
Shooter and Role-Playing games also perform strongly
Puzzle and Strategy games tend to have lower overall sales

4️⃣ Top Platforms by Global Sales

This visualization highlights the best-performing gaming platforms.
Top platforms typically include:
PlayStation 2
Xbox 360
PlayStation 3
Wii
Nintendo DS
These platforms contributed significantly to global game sales across multiple generations.

5️⃣ Top Games by Sales

A bar chart displays the best-selling games of all time, including titles such as:
Wii Sports
Grand Theft Auto V
Super Mario Bros.
Tetris

6️⃣ Regional Sales by Genre

Stacked bar charts compare regional preferences:
North America tends to favor Action and Shooter games
Europe shows similar trends but slightly stronger interest in Sports games
Japan has a stronger preference for Role-Playing games

7️⃣ Global Sales vs Regional Sales

A scatter plot visualizes relationships between:
North America sales
Europe sales
Game genre
This helps identify which genres perform best in specific markets.

🎯 Key Insights

The video game industry experienced rapid growth in the 2000s.
Action games dominate the market globally.
Console platforms like PS2 and Xbox 360 generated the highest total sales.
Regional preferences vary significantly, with Japan showing stronger demand for RPG titles.
A small number of blockbuster games account for a large portion of total sales.
